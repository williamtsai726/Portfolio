Projects & Clubs

Flower classification with Convolutional Neural Network (CNN)

Kaggle Dataset Source Code Skill: Python · tensorflow

Mobile applications that serve to classify flowers have been developed and this project aimed to simulate that classification functionality. Dataset of 102 types of flowers was taken from Kaggle and used to train the Convolutional neural network, which was expected to have the ability to classify 102 different types of flowers among the data.

German Traffic Sign with Convolutional Neural Network (CNN)

Kaggle Dataset Source Code Skill: Python · TensorFlow

Being able to recognize different traffic signs is essential to autonomous driving car. The project used data of 40+ different German Traffic signs taken at high speed to train the Convolutional neural network and test its accuracy. The model was then improved using Augmentation, Transfering and fine-tuning with VGG16.

Influenza trend prediction with Lstm, RNN, and GRU

CDC Record Report & Code Skill: Python · Pytorch

This project used the latest U.S. influenza dataset from the Center for Disease Control and Prevention to build RNN, LSTM, and GRU models and predict the influenza trends. The resulting predictions was then used to analyze and compare the effectiveness of individual models. The goal was to investigate which algorithm is better in solving this kind of linear regression problems.
